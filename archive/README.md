# Archive Folder

This folder contains files that are not part of the core AI Bug Bounty Hunter MCP Server project but are kept for reference or future use.

- **blog-files/**: Blog post drafts, guides, and screenshot generation utilities for blog content
- **screenshot-utilities/**: Scripts and guides for generating screenshots (not core to the MCP server functionality)
- **test-scripts/**: Experimental and test scripts used during development
- **misc/**: Miscellaneous documentation and files

## Note

These files are archived but preserved for reference. They do not affect the core functionality of the Bug Bounty MCP Server.
